frontend
 In order for this file (frontend) to work properly as your project’s frontend you must:

-Place it in the folder of your project.

-Download the ejs and axios modules ($ npm install ejs, $ npm install axios).

-Start the frontend server on port 9102 (command in terminal, in the project’s directory: “node frontend-index.js” or “nodemon frontend-index.js”), in a different terminal than your backend server (port 9103).
