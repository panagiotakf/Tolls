API-Backend
Για να λειτουργήσει το api-backend χρειαζόμαστε τα εξής αρχεία αποθηκευμένα στην ίδία διεύθυνση.
1) Το αρχείο index.js (έχει ανέβει στο φάκελο backend του github).
2) Έναν φάκελο που ονομάζεται endpoints και περιέχει τα διάφορα endpoints (adminhealthcheck,resetstations,passerperstatios κλπ.)
Τα endpoinds είναι αποθηκευμένα στο φάκελο api του github.
3) Έναν φάκελο που ονομάζεται key και περιέχει το certificate και το private key, απαραίτητα για το https.
Για περισσότερες πληροφορίες για αυτό χρήσιμο άρθρο εδώ: https://www.section.io/engineering-education/how-to-get-ssl-https-for-localhost/ πολύ χρήσιμο επίσης για πιστοποιητικά

4)Ένα αρχείο που ονομάζεται stations και πρέπει να είναι το excel αρχείο του βεσκούκη για τα stations
5) Το αντίστοιχο αρχείο για τα vehicles που επίσης ονομάζεται vehicles

Σημείωση: Στην ίδια διεύθυνση με τα παραπάνω είναι αποθηκευμένο και o φάκελος του frontend.
Για περισσότερες πληροφορίες σχετικά βλ. readme λευτέρη.
